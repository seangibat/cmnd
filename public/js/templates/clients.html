There is only one client right now: a terminal application written in Python. It should work in all Unix environments.

Download here: cmnd terminal client

Installation instructions:

  • tldr, in the same folder as cmnd:
    chmod 755 cmnd
    cp cmnd /usr/local/bin
    cmnd setup
  • Explanation:
  • Make the script executable: chmod 755 cmnd
  • Copy it to /usr/local/bin so it is executable anywhere: cp cmnd /usr/local/bin
  • Run cmnd setup to authenticate with cmnd's server.
  • That's it. Try running a few commands, if you've added an application.

In the future, hopefully, maybe, SDKs will be released for developing clients as well as applications.
